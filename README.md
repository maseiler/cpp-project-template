# C++ Project Template

## Prerequisites

- [CMake](https://cmake.org/)
- [Clang](https://clang.llvm.org/)

## Build and Run

Build

```bash
mkdir build && cd build
cmake ..
make
```

Run tests

```bash
./cpp-template-tests
```
